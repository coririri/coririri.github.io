# CodePen 알아보기
[Code Pen 사이트](https://codepen.io)  
간단하게 html, css, js를 테스트 해 볼수있는 사이트

## 유용한 정보
- HTML은 Body부분을 코딩하는 것
- 자바스크립트는 개발자 도구를 열어 확인
- 모든 영역은 각각 설정이 있고,  특히 js는 외부 라이브러리도 가져오기 가능
- CSS설정 -> CSS Base -> Reset설정
- 외부링크로 이동하기 위해선 ctrl+클릭 해야 한다.
